# Fissler eCommerce KPI Dashboard

## 프로젝트 개요
Fissler Korea의 전자상거래 채널(네이버 브랜드 스토어, 오픈 마켓 등)을 위한 KPI 대시보드입니다. 가상의 판매 데이터를 분석해 매출, 전환율, 평균 주문 가치를 시각화하며, 마케팅 및 운영 전략을 지원합니다.

## 기술 스택
- Python, Pandas, Streamlit, Plotly, SQLite
